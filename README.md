
# Vrinda Store Data Analysis Dashboard

This project presents an **interactive Excel dashboard** created for analyzing Vrinda Store sales data.  
The dashboard provides clear insights into sales performance, customer demographics, order behavior, and channel-wise distribution.

---

##Key Features

### ✔️ **Dynamic Filters (Slicers)**
- Month-wise filter  
- Category filter  
- Channel filter  

### ✔️ **Visual Insights**
- **Orders vs Sales** (Combo Chart)
- **Sales: Men vs Women** (Pie Chart)
- **Order Status Breakdown**
- **Top 5 States by Sales**
- **Orders: Age vs Gender**
- **Sales by Channels Distribution**

---

##Objectives
- Analyze high-performing states and categories  
- Identify which customer segment contributes most to sales  
- Monitor order trends across months  
- Understand channel-wise sales performance  
- Evaluate order status (Delivered, Cancelled, Returned, Refunded)

---

##Tools & Techniques Used
- Microsoft Excel  
- Slicers for interactive filtering  
- Combo Charts (Sales vs Orders)  
- Bar Charts  
- Pie Charts  
- Data Cleaning & Formatting  
- Pivot Tables and Pivot Charts  

---

##Files Included
- `Vrinda_Store_Dashboard.xlsx` – Final interactive dashboard  
- Additional sheets: Data cleaning, pivot tables, charts

---

##Insights Summary
- Women customers drive **98%** of total sales  
- Amazon & Flipkart are the top-performing channels  
- Karnataka, Maharashtra, and Tamil Nadu lead in sales  
- Order status shows **90% delivery success rate**
